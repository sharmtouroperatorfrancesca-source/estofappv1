ESTOF STAFF - GitHub build

COME FARE:
1. Crea un repository vuoto su GitHub
2. Carica TUTTO il contenuto di questo zip nel repository
3. Vai su Actions
4. Apri: Build ESTOF STAFF APK
5. Clicca Run workflow
6. Aspetta 2-4 minuti
7. In basso scarica l'artifact: estof-staff-apk
8. Dentro trovi: app-debug.apk

QUELLO è il file da installare sul telefono Android.

Se vuoi cambiare URL:
app/src/main/java/it/estof/staffapp/MainActivity.kt
